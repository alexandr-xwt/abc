ПРОБЛЕМЫ С НОВОСТЬЮ


1. новость может быть скриптом, требуется обработка htmlspecialchars.
    -> не делать
    -> текст предопределён
    -> ссылка предопределена
2. новость может содержать некорректно отображаемую страницу с html тэгами
    -> strip_tags()
    -> не делать
    -> текст предопределён
    -> ссылка предопределена
3. текст новости может быть рекламой, хамством, спамом
    -> требуется модерация
    -> не делать
    -> текст предопределён
    -> ссылка предопределена
4. кодировка страницы новости может отличаться от используемой в php
    -> проблема несущественна
    -> при переходе по ссылке браузер сам определяет кодировку
5. ссылка на новость может быть недоступна
    -> требуется система тестирования для сайта
    -> не делать
    -> не входит в ТЗ
